# Website Analysis

## Original Website
Sey Coffee

Website Link:
[https://examplewebsite.com](https://www.seycoffee.com/)

## Website Type
E-commerce Website

Website Interaction Analysis
Overview

This analysis evaluates the interaction design of the coffee shop website with the goal of identifying usability issues and opportunities to improve user engagement and conversion.

1. Call-to-Action Button ("Shop Coffees")
Problem

The "Shop Coffees" button currently appears static and blends into the minimalist background. Because of this, it does not strongly stand out as the primary Call to Action (CTA).

Users may overlook the button, reducing the likelihood that they proceed to browse products.

Proposed Improvement

Add a micro-interaction hover effect to increase visibility and interactivity.

Interaction Behavior

When the cursor approaches or hovers over the button:

The button scales to 1.05× its original size

Apply a 300ms ease-in transition

Expected Impact

This creates a subtle visual gravity effect, drawing the user's attention toward the button. It also confirms that the element is interactive and encourages users to explore the product catalog.

2. Product Grid Interaction
Problem

The product cards currently resemble static editorial images rather than interactive elements.

Because there is no clear interaction feedback, users may assume the cards are simply decorative images and scroll past them without clicking.

Proposed Improvement

Implement a Lift & Reveal interaction pattern.

Interaction Behavior

When the user hovers over a product card:

The card slightly lifts upward

A soft drop shadow appears

Product description text changes from light gray → bold black

A minimal "View Details" icon or "+" symbol appears in the corner

Expected Impact

This interaction provides clear clickability signifiers, transforming the product card from a passive image into an interactive element. This reduces user hesitation and increases the likelihood that users will explore individual products.

UX Principles Applied

This redesign focuses on improving:

Visual hierarchy

Interaction feedback

Affordance and clickability

User engagement

Conversion optimization
