# Coffee Shop Website Redesign (UI/UX Case Study)

## Project Overview
This project focuses on redesigning a coffee shop website with an emphasis on improving user experience, visual hierarchy, and conversion optimization.

The redesign addresses key usability issues identified in the original interface and introduces modern UI/UX principles to enhance user engagement and interaction.

---

## Original Website
A coffee shop website concept was analyzed and evaluated to identify usability and design issues affecting user interaction and conversion.

---

## Problems Identified
The following key issues were discovered during the analysis phase:

- Call-to-action buttons lacked visibility and interaction feedback  
- Product cards appeared static and not clearly clickable  
- Images were oversized, disrupting layout balance and alignment  
- Weak visual hierarchy affecting content readability  
- Lack of clear interaction cues for users  

---

## Design Solutions
To address these issues, the following improvements were implemented:

- Added **interactive CTA buttons** with hover animations to increase engagement  
- Redesigned **product cards** with hover effects (lift + reveal interaction)  
- Optimized **image sizes** to improve layout balance and consistency  
- Improved **visual hierarchy** through typography and spacing  
- Introduced **clear interaction cues** to reduce user hesitation

## Key UX Improvements
- Strong and clear **value proposition in the hero section**  
- Conversion-focused **"Shop Now" and "Buy Now" buttons**  
- Interactive **product grid with hover feedback**  
- Improved **layout alignment and spacing system**  
- Enhanced **user flow from browsing to action**
